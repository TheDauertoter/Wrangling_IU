# Wrangling_IU
Code for Project of IU, Data Wrangling

Task 1.3 - Tidy Up your Data

The Repo contains the .py File which is the code for the Task. To start, just clone the repo to a folder of your choice, install the requirements
and create an enviroment. 

To start - activate cleaning.py

raw_data - raw data of the coffe beans set as .xlxs

final_data - final table of the data after wrangling as .csv

Source of raw file : https://www.kaggle.com/datasets/saadharoon27/coffee-bean-sales-raw-dataset/data

This script performs data wrangling on a coffee sales dataset by loading multiple Excel sheets (orders, customers, and products), checking for data quality issues (missing values, duplicates, and format inconsistencies), and applying basic validations. The data is then cleaned, transformed, and merged into a single table, where additional features such as date components and aggregated sales metrics are calculated. The final output is a structured dataset ready for analysis.
